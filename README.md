三、工作、盈利方式
-----
:cyclone: 成员工作暂以当前工作为主，团队工作为辅，工作时间为当前工作的业余时间（工作时间也可以 :blush: ），工作方式为团队开发模式，使用OSChina上的Team把任务分好，使用GitHub为开发服务器以及Git团队开发管理工具，进行开发、签入、签出、提交，修改代码（所有大家还是要熟悉一下git工具的），每个项目分为参与者与辅助参与者。参与者为充分参与系统的分析开发测试等。辅助参与者为辅助系统开发成型以及工作量比较小的角色。

:cyclone: 团队中每个人都可以接项目，然后大家共同开发，团队处于空闲状态时，每个人都有责任来为团队推广(也就是接单子)，争取一步一步建立起属于我们团队的客户圈。这样通过为客户提供优质的服务，来创造我的价值。

四、盈利平分方式
-----
:cyclone: 我们的付出总会得到回报，我们将把每次奋斗盈利的10%作为工作室资金，其余的都作为团队成员所得。以加权平均方式发放给团队成员.（接单子的人可多得）
